How to Play

  2. If you spend 100 to trade, after deducting 5 service fee, your contract amount is 95.

  4. JOIN RED: if the result shows 2,4,6,8, you will get (95*2) 190; If the result shows 0, you will get (95*1.5) 142.5.
  5. JOIN VIOLET: if the result shows 0 or 5, you will get (95*4.5) 427.5.
